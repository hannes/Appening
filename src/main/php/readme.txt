How to setup Appening-PHP-Frontend:

0) Install MySQL (> 5.5), PHP (> 5.3)
1) Checkout Appening-PHP-Frontend

svn co https://subversion.assembla.com/svn/appening/Appening/src/main/php/ appening 

2) put this somewhere where the web server can run it
3) Put your MySQL pasword into helper.inc.php
4) Import data from server by running remote.sh [1]
5) Profit!

[1] https://www.assembla.com/code/appening/subversion/nodes/Appening/remote.sh
